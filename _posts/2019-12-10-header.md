To add a hero header to a page add the following to a YAML Front Matter:

```yaml
---
header:
  image: header.jpg
  background: "rgba(0, 0, 0, 0.5)"
  color: light
  title: How can we help you?
  subtitle:
  search: true
  ---
  ```
